FastFoodCompareAppEnhanced_v3.1 - ASP.NET Core MVC (.NET 6) sample project
- Uses EF Core InMemory provider for storing Dish and User data.
- Features: Login, Register, Session, Menu, Compare, Cart with quantity update, Ad placeholder, modern UI with hover effects.
- Default route: /Menu/Index
How to run:
1. Ensure .NET 6 SDK is installed.
2. Open this folder in Visual Studio or use command line:
   dotnet run
3. Open browser at https://localhost:5001 or http://localhost:5000 (as configured).
Notes:
- Replace images in wwwroot/images/ (burger.jpg, friedchicken.jpg, fries.jpg, salad.jpg, pizza.jpg, ads1.jpg).
- Default accounts: admin/123 and user/123
